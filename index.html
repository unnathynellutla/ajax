Studio Ghibli Movies

The API I chose was the Studio Ghibli API which gives data via HTTP request on Studio Ghibli movies, their directors descriptions, their runtimes, their characters, and all other information about the movies.

I found the api on the GitHub page in the readings that listed public API's. The link is here https://ghibliapi.herokuapp.com/#section/Studio-Ghibli-API.

I used the get method to get the data and XMLHttpRequest() to make the request as we did for most of the in class examples. I used request.open with get and the url and true to perform the request asynchronously.The data I selected was the titles of the movies and the descriptions.<\p>

This API would be helpful if you wanted to find common features between Studio Ghibli movies, or make a website with popular movies on it.<\p>
